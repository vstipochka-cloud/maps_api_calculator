About This Calculator

How It Works

This calculator helps you compare the cost of using different mapping API providers. It takes into account:

  • Free Tiers: Monthly free requests included with each provider
  • Volume Discounts: Lower prices at higher volumes (e.g., Mapbox)
  • New Customer Credits: Google's $300 trial credit for new accounts

Pricing Model

Provider allowances and tiers may differ: some providers assign per‑API free tiers while others use provider‑level allowances or plan‑specific bundles. Always consult the provider documentation for exact terms.

Technical Details

This calculator is built with:

  • Backend: Go REST API (running on http://localhost:8080)
  • Frontend: HTML/CSS/JavaScript (separate single-page app)
  • Real pricing data from official provider websites (March 2026)
  • CORS enabled for cross-origin requests
