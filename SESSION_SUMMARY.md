## 🎯 ЧТО БЫЛО СДЕЛАНО

### ✅ Исправлена КРИТИЧЕСКАЯ ошибка: HERE shared pool

**Проблема была:**
```
HERE имела 250K shared pool, но система применяла 
индивидуальные 30K free tiers на каждый API.

Input: geocoding 100K + routing 50K = 150K всего
Было:  (100K-30K)×0.75 + (50K-30K)×0.75 = 52.5 + 15 = $67.50 ❌
Стало: (150K-150K) = $0 ✅
```

**Тесты подтверждают:**
- HERE 150K (в пределах 250K) = $0 ✓
- HERE 300K (250K free + 50K paid) = $37.50 ✓  
- HERE 1M (250K free + 750K paid) = $562.50 ✓

### ✅ Реализована система `disable_free_tier`
- Можно отключить базовый free tier любого API
- Работает параллельно с `disable_new_customer_credit`
- **Примеры:**
  - `"query" = hide_free_tier": true` → платить за всё
  - `"disable_new_customer_credit": true` → без Google кредита

### ✅ Обновите модель данных
- Новая структура `MonthlyFreeTierInfo` для shared pools
- Различие между one-time кредитом и monthly free tier
- Готово для расширения на новых провайдеров
